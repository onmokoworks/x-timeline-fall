# X Timeline Physics

x.com / twitter.com の表示中タイムライン投稿を Matter.js の物理ボディにして、落下・衝突・マウスドラッグ・投げ操作で遊べる Chrome / Edge 用 MV3 拡張です。

## 使い方

1. Chrome または Edge で `chrome://extensions` を開く
2. 「デベロッパーモード」を有効化
3. 「パッケージ化されていない拡張機能を読み込む」
4. このフォルダを選択
5. x.com のタイムラインを開き、右下の `Drop` を押す

## 操作

- `Mode: Tweets`: 表示中のツイートだけを落とす
- `Mode: All`: ツイート以外のナビ、リンク、ボタン、画像、サイドバー部品も落とす
- `Drop`: 発動
- マウスドラッグ: 投稿を掴む
- ドラッグして離す: 投げる
- `Reset`: 元に戻す
- `Alt + Shift + P`: 発動 / リセット
- `Esc`: リセット

## 実装メモ

- 対象は発動時点で画面内に見えている要素です。
- x.com は仮想スクロールのため、無限にある全投稿を同時に物理化するより、表示中DOMだけを対象にした方が安定します。
- `All` モードでも親子要素の二重取得は避けています。DOM上の全ノードを同時に物理化すると、表示と当たり判定が破綻しやすいためです。
- Matter.js は `vendor/matter.min.js` に同梱しているため、実行時に外部CDNへアクセスしません。
